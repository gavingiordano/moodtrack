# moodtrack

## How to set up and run the web application:

### 1. Install the requirements via the command: pip install -r requirements.txt

### 2. Create a .env file in the root directory. Add a SECRET_KEY variable to something of your choosing.

### 3. Run the application from the root directory via the command: uvicorn app.main:app --reload

### 4. Navigate to http://127.0.0.1:8000/ to access the web app
